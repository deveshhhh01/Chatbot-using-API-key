🌐 Web Groq Chatbot (Python + Tkinter)

An Internet-aware AI chatbot built using **Python, Tkinter GUI, Groq API, and DuckDuckGo Search API**.

The chatbot can answer questions using:
- AI model (Groq Llama)
- Real-time web search results

It provides a simple **desktop chat interface** for interacting with the AI.

---

🚀 Features

- Modern **Tkinter GUI Chat Interface**
- AI responses powered by **Groq Llama-3 model**
- **Internet search integration** using DuckDuckGo API
- Handles API errors gracefully
- Clean UI with styled chat messages
- Lightweight desktop application

---

 🧠 Tech Stack

- Python
- Tkinter (GUI)
- Groq API (LLM)
- DuckDuckGo Instant Search API
- Requests Library
